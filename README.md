# 微信小程序：文字修仙（最小可运行版）

这是一个适合新手阅读的微信小程序放置修仙游戏。

## 如何运行

1. 打开微信开发者工具。
2. 点击“导入项目”。
3. 选择当前文件夹。
4. AppID 可使用测试号，或换成你自己的小程序 AppID。
5. 点击“编译”即可运行。

## 代码从哪里开始看

建议按下面的顺序阅读：

1. `pages/index/index.wxml`：页面上显示了什么。
2. `pages/index/index.js`：按钮、升级、突破、自动修炼和存档逻辑。
3. `pages/index/index.wxss`：页面样式。
4. `app.json`：小程序的全局配置。

## 当前数值规则

- 修炼一次：初始获得 1 点修为。
- 功法升级：初始消耗 20，之后每级费用翻倍；每级让点击修为 +1。
- 聚灵阵升级消耗灵石，费用公式为 `50 × (当前等级 + 1)²`；每级让自动修为 +1/秒。
- 外出历练每次有 60 秒冷却，可随机获得灵石或聚气丹。
- 聚气丹持续 60 秒，其间点击和自动修炼收益均为 1.5 倍。
- 境界包含凡人、炼气一至九层、筑基初中后期、金丹初中后期。
- 炼气九层突破筑基初期的成功率为 30%。
- 筑基后期突破金丹初期的成功率为 10%。
- 概率突破失败后，当前修为会全部清空。
- 所有进度使用 `wx.setStorageSync` 自动保存在本机。

> 微信开发者工具里可以通过“清缓存 → 清除数据缓存”删除存档并重新开始。
